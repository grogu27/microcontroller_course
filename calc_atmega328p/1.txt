Краткое описание работы:
При первом запуске:

Устанавливается пароль по умолчанию: "1234"

Сейф по умолчанию ЗАБЛОКИРОВАН

Сервопривод в положении "закрыто" (0°)

Для разблокировки:

Введите код "1234"

Сейф откроется, сервопривод повернется на 90°

В разблокированном состоянии:

Нажмите A для установки нового кода

Нажмите # для блокировки сейфа

В заблокированном состоянии:

Введите текущий код для разблокировки

Пароль сохраняется в EEPROM и будет работать после перезапуска симулятора.
